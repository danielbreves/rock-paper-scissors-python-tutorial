# Rock, Paper, Scissors - Python Tutorial

Learn Python by building a rock, paper, scissors game, one level at a time!

## Getting started

1. Click the green **Code** button at the top right of the page, then **Codespaces** > **Create codespace on main**
2. Wait for the codespace to load — it will open a code editor in your browser
3. On the left side you'll see a list of files — click on `level-1.py` to open it
4. Follow the instructions inside the file

## How it works

Each level teaches you something new. Read the instructions at the top of the file, write the code where it tells you, then run it to see what happens.

To run a level, type this in the **terminal** (the area at the bottom of the screen) and hit Enter:

```
python level-1.py
```

Replace the number with whichever level you're on.

## Levels

| Level | What you'll learn |
|-------|-------------------|
| 1 | Variables and `print()` |
| 2 | Getting input from the player |
| 3 | Lists, importing modules, and picking random values |
| 4 | Using `if` to check who wins |
| 5 | Using `elif` and `else` to combine checks |
| 6 | Using `while` loops to keep playing |
| 7 | Avoiding repeated code with variables |
| 8 | Handling invalid input |
| 9 | Organizing code with functions |
| 10 | Bonus: ideas to try on your own! |
